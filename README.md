1. Business Context
This project analyzes transactional data from Olist, a Brazilian marketplace connecting sellers and customers across multiple product categories.
The objective is to evaluate commercial performance, customer behavior, and operational efficiency in order to provide data-driven strategic recommendations.


2. Project Objectives
The project aims to:
Analyze revenue and growth dynamics
Evaluate customer value and retention
Identify top-performing product categories
Assess seller performance
Measure delivery performance impact
Build an executive dashboard in Microsoft Power BI


3. Dataset Overview
The dataset contains transactional data including:
Orders
Customers
Order items
Payments
Products
Sellers
Reviews
The data was cleaned, transformed, and modeled into an analytical schema to support KPI computation.
